# Dog Behavior Classifier

YOLO-based object detection model for detecting dog bathroom behaviors.

## Classes

- `dog_pooping`
- `dog_peeing`

## Model

Trained on YOLO11x with custom dataset.

- Input size: 1280x1280
- Format: ONNX

## Usage with Scrypted

Import this repository as a custom object detection model in Scrypted.
